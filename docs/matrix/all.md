## Overview

`matrix/all.hpp` includes the complete dense matrix module.

## Included Headers

| Header | Contents |
| --- | --- |
| `matrix/matrix.hpp` | Row-major dense matrices, arithmetic, multiplication, transposition, matrix-vector products, and powers. |
| `matrix/linear_algebra.hpp` | Gaussian elimination, rank, determinant, inverse, and linear systems. |
